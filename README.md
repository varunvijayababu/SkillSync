# SkillSync 🚀

### AI-Powered Resume Intelligence & Career Optimization Platform

SkillSync is a full-stack AI-powered career platform that helps users optimize resumes, analyze ATS compatibility, identify missing skills, and generate personalized career roadmaps.

Designed for students, freshers, and job seekers, SkillSync combines resume intelligence, recruiter simulation, and AI-assisted resume building into one seamless experience.

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/varunvijayababu/SkillSync.git
cd SkillSync
```

---

# 🔧 Frontend Setup

```bash
cd frontend
npm install
npm start
```

Frontend runs on:

```bash
http://localhost:3000
```

---

# 🔧 Backend Setup

```bash
cd backend
npm install
npm run dev
```

Backend runs on:

```bash
http://localhost:5000
```

---

# 🔑 Environment Variables

## Frontend `.env`

```env
REACT_APP_API_URL=your_backend_api_url
GENERATE_SOURCEMAP=false
```

---

## Backend `.env`

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
GROQ_API_KEY=your_groq_api_key
```
